Installation :
----------------------------------------------------
   Run : 
   --------------------------------------
       git clone https://github.com/ouhadjilyes/simple-newspaper-app.git
       cd <this repo>
       pipenv shell
       python3 manage.py runserver 
   -----------------------------------------
   then visit this link : 
   ------------------------------------
       http://127.0.0.1:8000/
   ------------------------------------
